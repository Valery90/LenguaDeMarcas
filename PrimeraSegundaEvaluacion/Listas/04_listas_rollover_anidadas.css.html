/********************************************************** Receta para listas rollover anidadas **********************************************************/ /* 1.-Haz una lista no ordenada de enlaces y métela en un <div id="navegacion">. Dentro de ella define otras del mismo tipo entre las etiqutas <li> y </li> Es importante por dos motivos: ->Generalmente este menú va en una caja en un lateral ->Queremos que el estilo no afecte a otras listas Todos los selectores empezarán por #navegacion 2.-Quita las "balas" */ #navegacion ul { list-style-type: none; } /* 3.-Como lo estamos haciendo "a mano" quitamos borde y padding. El problema es que son distintos en cada navegador. */ #navegacion ul { padding: 0; margin: 0; } /* 4.-Haz que los enlaces, que son elementos de línea, se muestren como bloques. */ #navegacion a { display: block; } /* 5.-Añade un color de fondo (y uno de texto que permita leer). */ #navegacion a { color: #FFF; background-color: #036; } /* 6.-Dale una anchura fija a los enlaces. */ #navegacion a { width: 9em; } /* 7.-Añade un padding */ #navegacion a { padding: 3px 12px 3px 8px; } /* 8.-Quita el subrayado de los enlaces. */ #navegacion a { text-decoration: none; } /* 9.-Cambia color y fondo para el estado hover. Recuerda que en CSS3 se pone :: */ #navegacion a:hover { background-color: #369; color: #FFF; } /* 10.-Separa un poco los elementos de lista. Hay problemas en algunos navegadores si se intenta hacer esto mediante márgenes. Es mejor jugar con bordes. Ponemos todos los márgener a cero y así se verá igual en todos los navegadores. */ #navegacion li { margin: 0; border-bottom: 1px solid #fff; } /* 11.-Sólo queda distinguir las listas visualmente. Nos copiamos las reglas anteriores y las modificamos para las listas internas. */ #navegacion li li a{ display: block; color: #FFF; background-color: #69C; /* cambiamos el color de fondo */ width: 9em; padding: 3px 3px 3px 17px; /* izquierda + derecha=20 como en la anterior */ text-decoration: none; border-bottom: 1px solid #fff; } /* 12.-Podemos resaltar la lista más externa en negrita. */ #navegacion a{ font-weight: bold; } /* 13.-Hay que definir otra regla para que las listas internas no salgan en negrita. */ #navegacion li li a{ font-weight: normal; }
