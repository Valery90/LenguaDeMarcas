Listas en HTML

En HTML tenemos tres tipos de listas

Listas no ordenadas

Las listas no ordenadas las realizaremos con la etiqueta <ul>

ul → unordered list

li → list item

  • Primer Elemento
  • Segundo
  • Tercero

No es recomendable pero podemos usar el atributo type


<ul type="disc"><!--"square" "circle" "disc" -->
	<li>Primer Elemento</li>
	<li>Segundo</li>
	<li>Tercero</li>
</ul>

Listas Ordenadas

Son las que en vez de un símbolo tienen a la izquierda de cada item un número o similar. Las construimos con <ol>

ol → ordered list

li → list item

  1. El uno
  2. El dos
  3. El tres
  4. otro más

Listas anidadas

  • Coffee
  • Tea
    • Black tea
    • Green tea
  • Milk

Date cuenta de que la segunda lista está en un elemento <li>


<ul>
  <li>Coffee</li>
  <li>Tea
    <ul>
      <li>Black tea</li>
      <li>Green tea</li>
    </ul>
  </li>
  <li>Milk</li>
</ul>

  • Tea

  • Milk
