Listas de HTML

En HTML tenemos tres tipos de listas

Listas no ordenadas

Las listas no ordenadas las realizaremos con la etqueta <ol>

ul → unordered list

li → list item

se puede
  • Primer Elemento
  • Segundo Elemento
  • Tercero Elemento

<ul type="circle">
<li>Primer Elemento</li>
<li>Segundo Elemento</li>
<li>Tercero Elemento</li>
</ul>
