Listas: Nociones básicas

En general se suelen usar tres propiedades para modificar las listas: list-style-type, list-style-position y list-style-image

Pongamos de lista de ejemplo los posibles valores de list-style-type

  1. armenian
  2. circle
  3. cjk-ideographic
  4. decimal
  5. decimal-leading-zero
  6. disc
  7. georgian
  8. hebrew
  9. hiragana
  10. hiragana-iroha
  11. inherit
  12. katakana
  13. katakana-iroha
  14. lower-alpha
  15. lower-greek
  16. lower-latin
  17. lower-roman
  18. none
  19. square
  20. upper-alpha
  21. upper-latin
  22. upper-roman

Define en firebug la regla ol { list-style-type: upper-alpha;} y mira las posibilidades.

A los símbolos que aparecen a la izquierda de cada elemento se les llama en inglés bullet(bala) y pueden cargarse desde un fichero.

Si quieres ver las consecuencias de las balas, define la regla ol { list-style-image: url('bala.jpeg');}.

Por último, podemos poner las viñetas en dos posiciones, dentro o fuera ol { list-style-position: outside; } (el valor por defecto es inside) .

Busca en w3schools los posibles atributos de <ol>, <ul> y <li>. Observa que atributos que antes se usaban están desaconsejados o directamente no funcionan en HTML5. Lo que sí es muy útil y hay gente que no sabe que existe son el atributo start de <ol> y el atributo value de <li>
